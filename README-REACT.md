# Portfolio_Module - React Version

Personal portfolio website for Ananya Saha - Built with **React + Vite**!

## Tech Stack

- React 18.2
- Vite 5.0
- Swiper.js for carousels
- Boxicons for icons

## Installation

Install dependencies:

```bash
npm install
```

## Running the Project

### Development Mode

```bash
npm run dev
```

Then open http://localhost:5173 in your browser.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- Fully responsive design
- Interactive hamburger menu
- Rotating text animation
- Tab-based About section
- Portfolio filtering
- Blog carousel with Swiper
- Animated skill progress bars
- Scroll progress indicator
- Smooth scroll navigation
